1 to initiate the variable used in the loop (i = 0). How ever this is not always tru, JS doesn't care. 1 is optional. You can initiate many values in statement 1 (separated by comma):.

2 is used to evaluate the condition of the initial variable. How ever this is not true, JS doesn't care. 2 is also optional. If statement 2 returns true, the loop will start over again, if it returns false, the loop will end.

3 increases the initial variable. How ever this his is not true, JS doesn't care, statement 3 is optional. 3 can do anything like negative increment (i--), positive increment (i = i + 15), or anything else. 3 can also be omitted (like when you increment your values inside the loop):.

Do/While loops

The do/while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.
